# Yik Yak Clone

A robust, AI-generated clone of Yik Yak with additional features like Business Ads, Ghost Posts, and Moderation.

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Anonymous Feed**: Location-based posts
- **Ghost Posts**: Time-limited content
- **Business Dashboard**: Ad management and analytics
- **Communities**: Join local herds
- **Moderation**: Auto-filter and manual reporting system

## Tech Stack

- **Framework**: Next.js 14+
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
